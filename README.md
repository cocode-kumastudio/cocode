## 快速开始 / Quick Start

### 克隆仓库 / Clone

```bash
git clone https://github.com/cocode-kumastudio/cocode.git
cd cocode
```

### 安装依赖 / Install Dependencies

```bash
composer install
```

### 启动开发 / Start Dev Server

```bash
php -S localhost:8000 -t public
```

> 具体项目请参考各子目录下的 README / Please refer to each subdirectory's README for specific projects.
