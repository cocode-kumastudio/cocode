## 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/cocode-kumastudio/cocode.git
cd cocode

# 2. 安装依赖
composer install

# 3. 启动开发服务器
php -S localhost:8000 -t public
```

> 具体项目请参考各子目录下的 README

## 如何参与

我们欢迎任何形式的贡献！

| 步骤 | 操作 |
|:---:|:---|
| **1** | 通过 [Issues](https://github.com/cocode-kumastudio/cocode/issues) 报告问题或提出建议 |
| **2** | Fork 仓库 → 创建分支（`feature/xxx` 或 `fix/xxx`）→ 开发 → 提交 PR |
| **3** | 在 [Discussions](https://github.com/cocode-kumastudio/cocode/discussions) 中交流想法 |

详细流程请阅读 [CONTRIBUTING.md](CONTRIBUTING.md)
