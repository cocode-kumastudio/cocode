### 简介

Cocode Hey3D 是一个**基于 AI 的 3D 模型生成工具**，用户只需上传照片，即可通过 AI 算法快速生成对应的 3D 模型。

Cocode Hey3D is an **AI-powered 3D model generation tool** that allows users to upload a photo and quickly generate a corresponding 3D model through AI algorithms.

### 技术架构

```
┌─────────────────────────────────┐
│         Electron Shell          │
├─────────────────────────────────┤
│        Vue 3 Frontend           │
│    (UI / 交互 / 模型预览)        │
├─────────────────────────────────┤
│       AI Inference Layer        │
│    (照片 → 3D 模型转换)          │
├─────────────────────────────────┤
│      File System / Export       │
└─────────────────────────────────┘
```

<br />

### 如何体验

1. 访问官网 [kuma2.cn](https://kuma2.cn) 下载最新版本
2. 安装并启动应用
3. 上传一张照片，等待 AI 生成 3D 模型
